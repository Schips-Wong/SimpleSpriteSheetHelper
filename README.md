# SpriteSheet 工具集

一个用于SpriteSheet图片处理的Python工具集，包含智能分割和重新对齐功能。

## 功能特性

### 分割工具
- 自动检测边界并分割SpriteSheet图片
- 自动计算可分割的图片数量
- 生成有序命名的图片文件
- 支持自定义输出目录

### 对齐工具
- 导入分割出来的小图片作为工作区
- 支持手动对齐精灵图的中心点
- 可视化工作区，支持拖拽调整
- 支持网格显示和中心点标记
- 重新拼接处理好的精灵图为完整精灵图表
- 支持行列数和间距调整

## 安装依赖

Python 3

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 图形界面分割工具

运行图形界面版本，具有实时预览功能：

```bash
python SpriteSheet2Sprite.py
```

**使用步骤：**
1. 点击"选择图片"按钮，选择要分割的SpriteSheet图片
2. 在"分割宽度"和"分割高度"输入框中设置每个小图片的尺寸
3. 实时预览区域会显示带分割线的图片效果
4. 可以选择输出目录，默认为`output`
5. 点击"开始分割"按钮，开始分割图片
6. 分割过程中会显示进度条，完成后会弹出提示窗口

### 2. 精灵图对齐工具

运行精灵图对齐工具，用于手动对齐分割后的小图片：

```bash
python Sprite2SpriteSheet.py
```

**使用步骤：**
1. 点击"导入分割图片"按钮，选择分割后的小图片（可多选）
2. 在左侧列表中选择要编辑的图片
3. 在工作区中可以：
   - 拖动图片调整位置
   - 使用右侧的X/Y偏移控件微调
   - 显示/隐藏网格和中心点标记
   - 重置偏移量
4. 设置拼接参数（列数、行数、间距）
5. 点击"拼接精灵图"按钮生成完整精灵图表
6. 点击"保存拼接结果"保存最终的精灵图

**对齐功能：**
- 工作区中央显示红色十字线和中心点
- 图片默认以中心点对齐
- 支持拖拽调整图片位置
- 支持精确的数值偏移调整
- 支持网格辅助对齐

## 注意事项

1. 确保安装了Python 3环境
2. 分割尺寸应小于或等于原始图片尺寸
3. 支持常见图片格式：PNG、JPG、BMP等
4. 输出图片默认保存为PNG格式
