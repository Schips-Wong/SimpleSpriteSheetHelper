# SpriteSheet 工具集

一个用于SpriteSheet图片处理的Python工具集，包含智能分割和重新对齐功能，支持中英文切换。

## 功能特性

### 分割工具
- 自动检测背景颜色和精灵区域
- 支持手动绘制和调整分割区域
- 可调整检测阈值以适应不同图片
- 支持缩放和平移查看
- 生成有序命名的图片文件
- 支持自定义输出目录
- 支持交换精灵之间的序号
- 中英文界面切换支持

### 对齐工具
- 导入分割出来的小图片作为工作区
- 支持手动拖拽调整位置
- 支持网格显示和中心点标记
- 参考图功能，支持半透明显示
- 支持自动对齐功能（左对齐、右对齐、上对齐、下对齐、中心对齐）
- 支持批量应用对齐设置
- 丰富的快捷键支持
- 支持缩放和平移查看
- 重新拼接处理好的精灵图为完整精灵图表
- 支持行列数和间距调整
- 支持导入多行目录，导出后每组一行
- 支持导入导出偏移设置
- 中英文界面切换支持

## 安装依赖

Python 3

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 精灵图分割工具

运行图形界面版本，具有自动精灵检测功能：

```bash
python SpriteSheet2Sprite.py
```

**使用步骤：**
1. 点击"选择图片"按钮，选择要分割的SpriteSheet图片
2. 点击"检测精灵"按钮，自动检测图片中的精灵区域
3. 可调整"自动检测阈值"以优化检测结果
4. 可手动添加、删除或调整分割区域
5. 支持Ctrl+滚轮缩放图片
6. 可以选择输出目录，默认为`output`
7. 点击"导出分割结果"按钮，导出分割结果图片
8. 分割过程中会显示进度条，完成后会弹出提示窗口

### 2. 精灵图组合工具

运行精灵图对齐工具，用于手动或自动对齐分割后的小图片：

```bash
python Sprite2SpriteSheet.py
```

**使用步骤：**
1. 点击"导入分割图片"按钮，选择分割后的小图片（可多选）
2. 在左侧列表中选择要编辑的图片
3. 在工作区中可以：
   - 拖动图片调整位置
   - 使用右侧的X/Y偏移控件微调
   - 显示/隐藏网格和中心点标记
   - 设置参考图，半透明显示以辅助对齐
   - 使用自动对齐功能快速对齐
   - 批量应用对齐设置到所有图片
4. 设置拼接参数（列数、行数、间距）
5. 点击"拼接并保存精灵图"按钮生成并保存完整精灵图表

**快捷键支持：**
- F5: 应用自动对齐到当前图片
- Ctrl+F5: 批量应用自动对齐到所有图片
- Delete: 删除当前图片
- Ctrl+': 切换网格显示
- Ctrl+S: 拼接并保存精灵图
- Q/W/A/S/D: 图片选择和上下移动
- C: 切换参考图显示
- H: 切换当前图片显示
- Alt+W/A/S/D: 微调X/Y偏移量

**对齐功能：**
- 工作区中央显示红色十字线和中心点
- 图片默认以中心点对齐
- 支持拖拽调整图片位置
- 支持精确的数值偏移调整
- 支持网格辅助对齐
- 参考图半透明显示，便于对比对齐
- 多种自动对齐方式可选

## 注意事项

1. 确保安装了Python 3环境
2. 支持常见图片格式：PNG、JPG、BMP等
3. 输出图片默认保存为PNG格式
4. 分割工具会自动检测精灵区域，无需手动设置分割尺寸
5. 对齐工具支持批量操作，可提高工作效率
6. 使用快捷键可以更快地完成操作
7. 参考图功能可帮助您更精确地对齐精灵图
